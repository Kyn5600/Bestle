# Security Policy

## Supported Versions

Bestle's versions will be serarated by version within the Bestle folder. The Weka folder contains the library. The Weka version will be the same for all iterations of Bestle.

| Version | Supported          |
| ------- | ------------------ |
| 1.0.0   | :white_check_mark: |
| 0.0.1   | :x:                |
